Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML includes all html tag like p tag , div tag,span tag and also contains text inside it. it will allow you to add html tag in new element and the webpage will provide this code accordingly.
innerText excludes all html tag , reads and modify only visible text inside them(doesn't read any text that are hidden by css) . example-

style {display : none } Document object model

it will not show p tag.. as it controlled by css
textContent - text content read and set all text including hidden text and doesnt shows any html tag. it has little better performance than
innertext as it doesnt consider any css class.

Question-3: What is event delegation in the DOM?

Event Delegation allows programmer to use one single event listener to parent element rather than adding
different event listener to each child.This parent element control all events for each of it's children element though it is newly added by createElements by js.

Question-4: What is event bubbling in the DOM?

Event bubbling define as a event that take place at child element
then like a bubble it trigger up to it's parent element ,then its parent
element and so on until it find the origin or root of dom.

Question-5: How do you create, add, and remove elements using JavaScript?

1. create element- by using createElement - const elementName = document.createElement('tagname' like p ,div etc); example : const p = document.createElement('p'); 2.add element : by using appendChild- const div = document.getElementById('div'); const p = document.createElement('p'); div.appendChild(p); 3.removeElement: by using removeChild- const parentDiv = document.getElementById('div'); parentDiv.removeChild('any existing tag')
